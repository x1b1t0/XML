Este repositorio documenta el proceso de aprendizaje de XML (Lenguaje de Marcas) visto en clase. Aquí se registrarán notas, ejercicios y ejemplos prácticos para seguimiento y revisión.

Objetivos
- Registrar conceptos y buenas prácticas de XML.
- Guardar ejemplos de documentos, esquemas (XSD) y transformaciones (XSLT).
- Documentar problemas comunes y sus soluciones.

Contenido
- /notes: apuntes y resúmenes de clase.
- /examples: documentos XML y casos de prueba.
- /schemas: XSD y archivos de validación.
- /xslt: hojas XSLT y resultados transformados.

Cómo usar
- Consultar /notes para teoría y definiciones.
- Ejecutar validadores y transformaciones sobre /examples.
- Añadir ejemplos nuevos siguiendo la estructura de carpetas.

Recursos
- Documentación oficial de W3C sobre XML.
- Material y prácticas proporcionadas por el profesor.

Contribuir
- Añadir o actualizar archivos en las carpetas correspondientes.
- Abrir issues para dudas, sugerencias o errores.
